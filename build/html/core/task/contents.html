task

class task(mutiprocessing.Process)

This class define a background task, it inherits from multiprocessing.Process. It provides a friendly API to run a given function in background

__init__(function, name, args=[], kwargs={})

The constructor inits the task with the target function, and inits the attributes.

Parameters:

  • function (function) – The function to run in background

  • name (str) – The name of the current task

  • args (list) – The list of unamed arguments

  • kwargs (dict) – The dictionary of named arguments

run()

The method runs the target function in background.

The standard output is automatically redirected to a temporary file, named <taskName>-<taskPID>.out.

stop()

The method stops the current task.

start()

The method starts the current task.

toList()

The method returns a list representing the current task. The list is [PID, name, state, outputFileName]

Returns:

list representing the current task

Return type:

list or str

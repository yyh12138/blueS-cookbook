class Module

This class defines the standard behavior of a module. It is the base class for all modules.

__init__()

This constructor inits the module.

out(success, output={})

This method help to format the output of the module.

Parameters:

  • success (bool) – True if the module has been executed successfully, False otherwise.

  • output (dict of str) – The output of the module.

Returns:

The formatted output of the module, like {“success”: success, “output”: output}

Return type:

dict

ok(output={})

This method is a shortcut to call out() with success=True.

Parameters:

output (dict of str) – The output of the module.

Returns:

The formatted output of the module, like {“success”: True, “output”: output}

Return type:

dict

nok()

This method is a shortcut to call out() with success=False.

Returns:

The formatted output of the module, like {“success”: False, “output”: {}}

Return type:

dict

init()

It must be overloaded in order to define the parameters of module

prerun()

It must be overloaded in order to define the actions to do before running the module

run()

It must be overloaded in order to define the actions to do when running the module

postrun()

It must be overloaded in order to define the actions to do after running the module

execute()

This method is the main method to call to run the module. It calls prerun(), run() and postrun() in the right order.

Raises:

  • KeyboardInterrupt – If the user wants to stop the execution of the module

  • EOFError – If the module is not correctly initialized

info()

This method helps to format the output of the module

Returns:

The formatted output of the module, like {“name”: name, “type”: type, “description”: description, “dependencies”: dependencies}

Return type:

dict

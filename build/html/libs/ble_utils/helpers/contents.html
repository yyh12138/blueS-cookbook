helpers

This module helps to manipulate BLE link layer packets.

frequentToChannel(frequency)

Convert frequency to channel number.

Parameters:

frequency (int) – Frequency in MHz.

Returns:

Channel number.

Return type:

int

Example:
>>> frequentToChannel(2402)
37

crc24(data, length, init=0x555555)

Calculate CRC24 of the given data.

Parameters:

  • data (bytes) – Data to calculate CRC24.

  • length (int) – Length of the data.

  • init (int) – Initial value of the CRC24.

Returns:

CRC24 value.

Return type:

bytes

Example:
>>> data = bytes.fromhex("0215110006000461ca0ce41b1e430559ac74e382667051")
>>> crc24(data, len(data)).hex()
"545d96"
isAccessAddress(aa)

Check if the given value is an access address

  • bits cannot be all 0 or all 1

  • every 6 bits cannot be all 0 or all 1

  • last 2 bits cannot be all 1

Parameters:

aa (int) – Value to check.

Returns:

True if the value is an access address, False otherwise.

Return type:

bool

Example:
>>> isAccessAddress(0x8E89BED6)
True

setAttribute(handle=None, value=None, type=None, permissions=None)

Add an attribute to the database

Parameters:

  • handle (int) – The handle of the attribute

  • value (bytes) – The value of the attribute

  • type (int or bytes or hex or string) – The type of the attribute

  • permissions (int or list or string) – The permissions of the attribute

Example:
>>> db = ATT_Database()
>>> db.setAttribute(handle=1, value=b'\x00\x18', type="Primary Service", permissions=["Read"])
>>> db.show()

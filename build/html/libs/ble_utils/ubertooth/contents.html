class BLEUbertoothDevice(BtUbertoothDevice)

This class represents a BLE device by Ubertooth, which inherit the BtUbertoothDevice

getAccessAddress()

This method gets the access address of the BLE device

setSweepingMode(enable=True, sequence=[37, 38, 39])

This method sets the sweeping mode of the Ubertooth device. It allows to provide a subset of advertising channels to monitor sequentially

Parameters:

  • enable (bool) – Enable or disable the sweeping mode

  • sequence (list) – The sequence of the channels to sweep

setJamming(enable=True)

This method sets the jamming mode of the Ubertooth device.

Parameters:

enable (bool) – Enable or disable the jamming mode

This method restarts the sniffing mode of the Ubertooth device by sniffNewConnections or sniffExistingConnections

This method sniffs the advertisements of the BLE device

Parameters:

  • channel (int) – The channel of the BLE device

sniffNewConnections(address='00:00:00:00:00:00', channel=None)

This method sniffs the new connections of the BLE device

Parameters:

  • address (str) – The address of the BLE device

  • channel (int) – The channel of the BLE device

sniffExistingConnections(accessAddress=None, crcInit=None, channelMap=None)

This method sniffs the existing connections of the BLE device

Parameters:

  • crcInit (int) – The CRC initialization value of the BLE device

  • channelMap (int) – The channel map of the BLE device

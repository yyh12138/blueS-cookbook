start8File(filename=None, loop=True)

This method starts to jam BLE channels by configure file

Parameters:

  • filename (str) – The filename to play

  • loop (bool) – The loop to set

jam8Channels(channels, power, PSR, length)

This method jams BLE channels

Parameters:

  • channels (list) – The channels to jam. It ranges [1, 14]

  • power (int) – The power to jam. It ranges [-128, 127]

  • PSR (int) – The Packets Send Ratio to jam. It ranges (0, 1]

  • length (int) – The length to jam. It ranges [0, 1500]

jamAllChannels(power, PSR, length)

This method jams all BLE channels by jam8Channels

Parameters:

  • power (int) – The power to jam.

  • PSR (int) – The Packets Send Ratio to jam.

  • length (int) – The length to jam.

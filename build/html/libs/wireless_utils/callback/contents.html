callback

class Callback

This class is a callback class that can be linked to an event

__init__(event='*', function=None, args=[], kwargs={}, background=True)

This method initializes the callback. Refer to the libs.wireless.Receiver.onEvent() method for more information

Parameters:

  • event (str) – the format how the event received

  • callback (function) – the callback to attach

  • args (list) – the unamed arguments to pass to the callback

  • kwargs (dict) – the named arguments to pass to the callback

  • background (bool) – if True, the callback will be executed in background

update(packet)

This method updates the callback runnable attribute if the packet matches the event defined

run(packet)

This method runs the callback if the packet matches the event defined

Parameters:

packet (blues.libs.wireless_utils.packets.Packet) – the packet to check

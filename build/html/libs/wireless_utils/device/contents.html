device

class Device

This class communicates with a hardware device. Each device must inherit from this class, and implement the following methods: init(), isUp(), send(packet)(), recv(), close().

Each device has a unique name, which is a string stored in INTERFACE attribute

__init__(self, interface)
Parameters:

interface (str) – The name of the device

get(cls, interface)

This method returns the only one instance with the given name

Parameters:

interface (str) – The name of the device

Return Device:

The device with the given name

Return type:

Device

subscribe(subscriber)

This method registers a subscribing function to the device

Parameters:

subscriber (function) – The subscribing function

publish(event, *args, **kwargs)

This method publishes an event, if the emitter/receiver subscribe and implements the event

Parameters:

  • event (str) – The name of the event

  • args (str) – The unamed arguments of the event

  • kwargs (dict) – The named arguments of the event

hasCapabilities(*capability)

This method checks if the device has the given capabilities

Parameters:

*capability (str) –

The name of the capability

Return bool:

if the device has the given capabilities

Example:
>>> device.capabilities = ['SNIFFING', 'CHANGING_ADDRESS', 'CONNECTING']
>>> device.hasCapabilities('SNIFFING', 'CHANGING_ADDRESS')
True
isUp()

This method checks if the device is up. It must be implemented by the child class

Return bool:

if the device is up

init()

This method initializes the device. It must be implemented by the child class

send(data)

This method sends data to the device. It must be implemented by the child class

Parameters:

data (bytes) – The data to send

recv()

This method receives data from the device. It must be implemented by the child class

Return bytes:

The received data

close()

This method closes the device. It must be implemented by the child class

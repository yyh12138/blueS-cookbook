class BtHCIDevice(wireless.Device)

This class is a device that can communicate with an HCI device to use Bluetooth. The interface is hciX where X is the number of the device

__init__(interface)
Parameters:

interface (str) – The interface of the device

init()

This method init the bluetooth device by send HCI CMD

_createSocket()

This method create a bluetooth user socket (from scapy.all.BluetoothUserSocket) to communicate with the HCI device

Returns:

If the socket is created

Return type:

bool

Raises:

BluetoothSocketError – If the socket can’t be created

isUp()

This method check if the bluetooth device is ready

send(data)

This method use socket to send a packet to the HCI device

Parameters:

data (scapy cmd packet) – The packet to send

recv()

This method use socket to receive a packet from the HCI device

Returns:

The packet received

Return type:

scapy cmd packet

Raises:

OSError – If the socket can’t find a socket

_internalCommand(cmd, noResponse=False)

This method send a command to the HCI device and wait for the response

Parameters:

  • cmd (scapy cmd packet) – The command to send

  • noResponse (bool) – If the command don’t need a response

Returns:

The response

Return type:

scapy cmd packet

Raises:

BluetoothCommandError – If the response is an error

onEvent(event='*', callback=None, args=[], kwargs={}, background=True)

This method attach a callback to an reveiving event, triggered when receive some special blues packets

Parameters:

  • event (str) – the format how the event received

  • callback (function) – the callback to attach

  • args (list) – the unamed arguments to pass to the callback

  • kwargs (dict) – the named arguments to pass to the callback

  • background (bool) – if True, the callback will be executed in background

Example(event):

  • *: each packet received will trigger the callback

  • n: every n packets received will trigger the callback

  • packetType: each packetType received will trigger the callback

Example:
def show(packet, value):

io.info(“The receiving packet: “ + packet) io.info(“The value: “ + value)

>>> receiver.onEvent("BLEConnectRequest", callback=show, args=["yyy"]) # show the packet and value when a BLEConnectRequest received
